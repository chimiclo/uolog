## サービス概要
UOLOGは、釣り人が「釣った魚の記録を残し、見返し、共有して上達に繋げる」ための釣果ログサービスです。
写真・場所・魚種・天候などの詳細なデータを蓄積し、自分だけの”釣りノート”を作れます。
さらに、他の釣り人の釣果も見られ、コミュニティとして楽しむこともできます。

## このサービスへの思い・作りたい理由
釣りは経験がものを言う趣味です。
しかし、釣行記録を残さずにいると「どの季節、どのポイントでどんな仕掛けが効いたか」を忘れてしまい、成長を実感し難いです。
- SNSは「見せるための投稿」が中心
- 個人だけでつける釣行日記は面倒で続かない
- 釣り情報は流れていくだけで資産化されない
以上の課題を解決するため、UOLOGは、『釣果の記録×振り返り×共有による上達体験』を中心に設計しています。
釣って終わりではなく、自分の釣りが進化していく過程が見えるアプリにしたいと言う思いから生まれました。

## ユーザー層について
釣り初心者〜中級者、釣りを日常的に楽しむ人、SNS投稿を目的にしない釣り人。
承認欲求よりも「釣りの深まり」を求めている方、過去の釣果を振り返りたい、知見を蓄積したい方をターゲットとしています。

## サービスの利用イメージ
1.釣ったら写真を撮り、UOLOGにそのまま登録
2.記録は自分の「釣りデータベース」として蓄積
3.過去データを振り返れば季節・天候・仕掛けの傾向が見える
4.他ユーザーの記録から学んで、次回の釣行に活かせる
→ 自然と「成長のループ」が回り続ける設計です。

## ユーザーの獲得について
まずはアプリ開発者自身の釣りコミュニティ（友人や家族）への共有、
その後Instagram / X の釣果タグ投稿者へアプローチを行います。

## サービスの差別化ポイント・推しポイント
一般的な釣りSNS投稿との差別化ポイントとしては
- 写真/ポイント/魚種/天候/仕掛け/サイズなど、記録項目が詳細。
- 分析、統計、振り返りができる学習性。
- 情報交換、ノウハウ共有が中心でコミュニティとして繋がる。
記録 × 分析 × 成長 を軸に置いている点が最大の推しポイントです。
